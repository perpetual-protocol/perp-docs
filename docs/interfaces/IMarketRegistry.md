### getFeeRatio
```solidity
  function getFeeRatio(
    address baseToken
  ) external returns (uint24 feeRatio)
```

The ratio is in `1e6` format, that means `1% = 1e4`

#### Parameters:
| Name                           | Type          | Description                                                                  |
| :----------------------------- | :------------ | :--------------------------------------------------------------------------- |

#### Return Values:
| Name                           | Type          | Description                                                                  |
| :----------------------------- | :------------ | :--------------------------------------------------------------------------- |
|`feeRatio` | uint24 | The fee ratio of the market, it is a decimal in `1e6`
